# E-commercesales-analysis
E-commerce sales data analysis project using Python (Pandas, Matplotlib) with insights on sales trends, category performance, and profit relationships.
## Business Insights
- Electronics category generates highest revenue
- Sales show an increasing trend over time
- Profit is positively correlated with sales
- Furniture has moderate sales but stable profit
